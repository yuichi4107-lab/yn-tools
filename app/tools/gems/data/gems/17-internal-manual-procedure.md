# 社内マニュアル・手順書作成 — Gemini Gem

## 基本情報
- **カテゴリ**: 報告書・ドキュメント
- **対象ユーザー**: 会社員（AI初心者含む）
- **想定利用シーン**: 業務手順をマニュアル化する、新しく導入したツールの操作手順書を作る、属人化している業務を文書化する

## Gem 設定

### Gem名
社内マニュアル・手順書メーカー

### 説明文
業務の流れやツールの操作手順を伝えるだけで、誰でも迷わず作業できるマニュアル・手順書を作成します。属人化した業務の見える化に最適です。

## 使い方ガイド

### こんなときに使えます
- 自分にしかできない業務を、チームの誰でもできるようにしたいとき
- 新入社員や異動者向けに業務手順を文書化したいとき
- 新しいツールやシステムを導入し、操作マニュアルが必要なとき
- 既存のマニュアルが古くなったり分かりにくくなったりしたとき

### 使い方のコツ
- **口頭で説明するように話す**: 「まずExcelを開いて、次にこのシートのA列に日付を入れて...」のように、やっている順番で話せばOKです
- **読者を明確にする**: 「PCに詳しくない人向け」「新卒向け」など読者を指定すると、説明の粒度が最適化されます
- **例外パターンも伝える**: 「通常はAだけど、月末だけBになる」といった例外は重要な情報です
- **段階的に作る**: まず大枠を作ってもらい、その後で各手順を詳細化していく方法が効率的です

### 入力例

**入力:**
```
会議室予約の手順書を作りたいです。新入社員向けで。

手順:
- 社内ポータルにログイン
- 「施設予約」メニューを開く
- カレンダーから希望日時を選ぶ
- 会議室を選んで予約ボタンを押す
- 参加者にメールで通知が自動で飛ぶ

注意:
- 大会議室（20名以上）は部長の承認が必要
- キャンセルは前日17時まで
- 予約が重複すると先着順になる
```

**出力例:**
```
# 会議室予約 手順書
最終更新日: 2026年3月14日
作成者: （要記入）
対象読者: 新入社員
バージョン: 1.0

## 1. はじめに
この手順書では、社内の会議室を予約する方法を説明する。
予約はすべて社内ポータルサイト上で行う。

## 2. 事前準備
- 社内ポータルサイトのアカウント（入社時に付与済み）
- ログインID・パスワード（不明な場合は情報システム部に問い合わせる）

## 3. 手順

### 手順1: 社内ポータルにログインする
1. ブラウザで社内ポータルサイト（URL: 要確認）を開く
2. ログインIDとパスワードを入力し、「ログイン」ボタンをクリックする
※ここにスクリーンショットを挿入

### 手順2: 施設予約メニューを開く
1. トップページ上部のメニューバーから「施設予約」をクリックする
※ここにスクリーンショットを挿入

### 手順3: 希望日時を選択する
1. カレンダーが表示されるので、予約したい日付をクリックする
2. 時間帯を選択する（開始時刻と終了時刻を指定）

### 手順4: 会議室を選択・予約する
1. 空いている会議室の一覧が表示される
2. 利用したい会議室を選び、「予約」ボタンをクリックする
3. 予約内容の確認画面が表示されるので、内容を確認して「確定」をクリックする

【注意】大会議室（20名以上の部屋）を予約する場合は、部長の事前承認が必要である。予約前に部長へ口頭またはメールで承認を得ること。

### 手順5: 予約完了を確認する
1. 「予約が完了しました」という画面が表示される
2. 参加者には自動でメール通知が送信される

## 4. よくあるトラブルと対処法

| 症状 | 原因 | 対処法 |
|---|---|---|
| 希望の会議室が選択できない | 他の人がすでに予約済み | 別の時間帯・会議室を検討する。先着順のため早めの予約を推奨 |
| 予約ボタンが押せない | 大会議室の承認が未完了 | 部長の承認を取得してから再度操作する |

## 5. FAQ
- **Q: 予約のキャンセルはいつまで可能？**
  → 前日17:00まで。施設予約メニューの「予約一覧」からキャンセル可能。
- **Q: 定期的な会議（毎週同じ時間）をまとめて予約できる？**
  → 要確認（繰り返し予約機能の有無を管理者に確認すること）

## 6. 問い合わせ先
- 予約システムの操作に関する問い合わせ: 情報システム部（内線: 要確認）
- 会議室の設備トラブル: 総務部（内線: 要確認）
```
