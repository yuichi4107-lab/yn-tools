# 業務引き継ぎ資料作成 — Gemini Gem

## 基本情報
- **カテゴリ**: 報告書・ドキュメント
- **対象ユーザー**: 会社員（AI初心者含む）
- **想定利用シーン**: 異動・退職・産休育休前の引き継ぎ資料を作る、担当変更時に後任者向けの資料を準備する

## Gem 設定

### Gem名
業務引き継ぎ資料アシスタント

### 説明文
異動や担当変更の際に必要な引き継ぎ資料を、漏れなく分かりやすく作成します。業務内容を伝えるだけで、後任者がすぐ動ける資料に仕上げます。

### インストラクション

## 使い方ガイド

### こんなときに使えます
- 異動や退職が決まり、短期間で引き継ぎ資料を準備する必要があるとき
- 担当業務が多すぎて、何から書けばいいか分からないとき
- 口頭だけの引き継ぎでは不安で、文書として残しておきたいとき
- 産休・育休に入る前に、漏れのない引き継ぎ資料を作りたいとき

### 使い方のコツ
- **まずは業務を箇条書きで列挙する**: 完璧でなくて構いません。まずは担当業務の一覧を出すと、Gemが整理してくれます
- **対話形式で進める**: 一度にすべての情報を入力しなくても、Gemが質問してくれるので、会話しながら資料を完成させられます
- **トラブル事例を忘れずに**: 「過去にこんなミスがあった」「ここは特に注意」という情報が、後任者にとって最も価値があります
- **最後に目次を確認**: 完成した資料の目次を見て、抜け漏れがないかチェックしましょう

### 入力例

**入力:**
```
4月から異動になるので、引き継ぎ資料を作りたいです。

担当業務:
- 毎月の売上レポート作成（月初3営業日以内、Excelで作成→部長に提出）
- 取引先C社との定例会議（月1回、第2水曜14時）
- 社内備品の発注管理（在庫少なくなったら都度発注、Amazon Businessで購入）
- 新入社員研修の資料準備（毎年3月、人事部の山田さんと連携）

注意点:
- 売上レポートは営業部の佐藤さんからデータをもらう必要がある。催促しないと遅れがち。
- C社の担当者は4月に変わる可能性あり
```

## 2. 業務詳細

### 売上レポート作成
- **概要**: 前月の売上実績をExcelでまとめ、部長へ提出する
- **頻度**: 毎月月初（3営業日以内）
- **手順**:
  1. 月初1営業日目に、営業部 佐藤さんへ売上データの提供を依頼する
  2. データを受領したら、所定のExcelテンプレートに入力する
     - テンプレート保存場所:（要確認）
  3. 前月比・前年比のグラフが自動更新されることを確認する
  4. 完成したレポートを部長へメールで提出する
- **注意点**:
  - 佐藤さんからのデータ提供が遅れることがある。月末時点でリマインドメールを送っておくとスムーズ
  - 数値の桁間違いが過去にあったため、提出前に必ず前月データと突合する

### C社 定例会議
- **概要**: 取引先C社との月次定例会議に出席し、進捗共有・課題協議を行う
- **頻度**: 毎月第2水曜 14:00〜（約1時間）
- **手順**:
  1. 会議の1週間前に議題を社内で取りまとめる
  2. 前日までにC社担当者へ議題を共有する

  4. 議事録を社内関係者とC社へ共有する
- **関係者**: C社 担当者（※2026年4月に担当変更の可能性あり。新担当者は要確認）
