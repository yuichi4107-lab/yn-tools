# グラフ・チャート選定・作成指示 — Gemini Gem

## 基本情報
- **カテゴリ**: Excel・データ業務
- **対象ユーザー**: 会社員（AI初心者含む）
- **想定利用シーン**: データを視覚的にわかりやすく伝えたいとき。どのグラフが適切かわからない、見栄えのいいグラフを作りたい場面

## Gem 設定

### Gem名
グラフ・チャート選定ガイド

### 説明文
データの内容と伝えたいメッセージを伝えると、最適なグラフの種類を選び、Excelでの作成手順を具体的に指示します。見やすいデザインのコツも教えます。

## 使い方ガイド

### こんなときに使えます
- 報告書やプレゼン資料にグラフを入れたいとき
- 円グラフか棒グラフか迷っているとき
- 作ったグラフが見づらいので改善したいとき
- 経営層向けに一目で伝わるグラフを作りたいとき
- 複数のデータを1つのグラフにまとめたいとき

### 使い方のコツ
- **「何を伝えたいか」を先に伝える**：同じデータでも「推移を見せたい」なら折れ線、「比較したい」なら棒グラフと最適解が変わります
- **データの列名と行数を伝える**：「月・商品A・商品B・商品C の4列、12ヶ月分」のように構造を伝えると的確な提案が返ってきます
- **相手に合わせる**：「社長向けプレゼン」「チーム内共有」など、読み手を伝えるとデザインの方向性が変わります
- **既存グラフの改善も相談OK**：「今こんなグラフなんですが見づらくて...」と現状を伝えれば、改善案を出してくれます

## 推奨グラフ：複合グラフ
- 売上実績 → 棒グラフ（メインのデータとして目立たせる）
- 売上目標 → 折れ線グラフ（基準線として重ねる）

この組み合わせにすると、棒が折れ線を超えている月＝目標達成、
棒が折れ線を下回っている月＝未達、が直感的にわかります。

## Excelでの作成手順

### ステップ1：データ範囲を選択
- A1:C13（月・売上実績・売上目標の12ヶ月分）を選択

### ステップ2：グラフの挿入
1. 「挿入」タブをクリック
2. 「おすすめグラフ」→「すべてのグラフ」タブ
3. 左メニューから「組み合わせ」を選択
4. 「売上実績」→「集合縦棒」を選択
5. 「売上目標」→「折れ線」を選択 → 「OK」

### ステップ3：デザイン調整
1. 売上実績の棒の色 → 青系の落ち着いた色（部長向けなので派手すぎない）
2. 売上目標の折れ線 → 赤の点線（基準線として目立たせすぎない）
3. データラベルは棒グラフにのみ表示（数値を見やすく）
4. 凡例をグラフの上部に配置

### 見やすくするコツ
1. Y軸の最小値を0にする（データを歪めない）
2. グリッド線は薄いグレーにして目立たせない
3. タイトルは「2025年度 月別売上実績 vs 目標」のように具体的に
4. 目標達成月と未達月で棒の色を変えると、さらにわかりやすい
   （条件付き書式はグラフには使えないため、VBAまたは手動で色分け）

---

3D効果やグラデーションは使わないでください。
見た目は派手になりますが、数値の正確な読み取りを妨げます。
